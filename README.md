# Smart Grievance Management System

## Overview

The **Smart Grievance Management System** is a web-based application that allows citizens to submit complaints and track their status efficiently.
The system automatically routes complaints to relevant departments and enables administrators to manage and resolve them effectively.

---

## Features

* 📝 Complaint Registration with file upload
* 🔍 Complaint Tracking System
* 🧠 Duplicate Complaint Detection
* ⚡ Priority-Based Complaint Handling
* 🏢 Department Routing
* 📊 Admin Dashboard
* 📧 Email Notification System
* 🗑️ Complaint Deletion Feature

## Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** Python (Flask)
* **Database:** SQLite
* **Deployment:** PythonAnywhere
* **Version Control:** Git & GitHub

## ⚙️ Installation (Local Setup)

### 1. Clone the repository

```bash
git clone https://github.com/manojkumarmanub-CV/smart-grievance-system.git
cd smart-grievance-system
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate environment

```bash
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the application

```bash
python app.py
```

## 📧 Email Configuration

Create a `.env` file and add:

```env
SECRET_KEY=your_secret_key
EMAIL_ADDRESS=yourgmail@gmail.com
EMAIL_PASSWORD=your_app_password
```

⚠️ Use **Gmail App Password** instead of your normal password.
